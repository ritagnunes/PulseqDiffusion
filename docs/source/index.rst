PulseqDiffusion
=================

.. image:: https://img.shields.io/badge/made%20using-pypulseq-brightgreen
   :align: left

.. image:: ../../Logo.png
   :align: center

`PulseqDiffusion <https://github.com/ritagnunes/PulseqDiffusion>`_ enables vendor-neutral design of multi-slice single-shot spin-echo echo-planar imaging (EPI)-based diffusion pulse sequences supporting multiple b-values and directions [1]_, using `PyPulseq <https://github.com/imr-framework/pypulseq>`_ [2]_, a Python translation of the Pulseq framework originally written in Matlab [3]_.

.. [1] Nunes, Rita G., et al. "Implementation of a Diffusion-Weighted Echo Planar Imaging sequence using the Open Source Hardware-Independent PyPulseq Tool", 28th Annual Meeting of the International Society for Magnetic Resonance in Medicine, Sidney, Australia (2020).

.. [2] Ravi, Keerthi Sravan, et al. "Pulseq-Graphical Programming Interface: Open source visual environment for prototyping pulse sequences and integrated magnetic resonance imaging algorithm development." Magnetic resonance imaging 52 (2018): 9-15.

.. [3] Layton, Kelvin J., et al. "Pulseq: a rapid and hardware‚Äêindependent pulse sequence prototyping framework." Magnetic resonance in medicine 77.4 (2017): 1544-1552.
